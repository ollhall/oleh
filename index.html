Вітаю Вас на своєму сайті-портфоліо. Мене звати Олег. Займаюся HTML-версткою сайтів. У мене повністю кросбраузерна, адаптивна та валідна верстка. Для забезпечення виконання вимог, що пред'являються до верстки сайту в процесі роботи над проектом, я використовую збирач Gulp для автоматизації процесу верстки, препроцесори Less/Sass для прискореної стилізації сайту та більш гнучкого коду, БЕМ та media-запити для адаптивної верстки сайту. Динаміка сайту організується власним JavaScript кодом, бібліотеки jQuery або налаштованими плагінами. Наприкінці роботи здійснюється перевірка відображення верстки у різних браузерах на різних пристроях, код перевіряється валідатором на відповідність стандартам та усуваються виявлені помилки. Можлива інтеграція верстки до CMS WordPress.
